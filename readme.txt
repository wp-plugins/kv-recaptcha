== Description ==

WordPress KV reCaptcha plugin will help you to filter spam registrations and help you to prevent brute force attack


= Features =
* Enable disable options
* you can extend it to use the recaptcha for comment form and contact form. 
* Simple Set-up, Less space consuming


 

== Installation ==

1. Upload `kv_recaptcha` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
